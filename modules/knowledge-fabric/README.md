# Knowledge Fabric

> Cross-domain contextual memory layer.

This module provides the unified knowledge layer that connects insights across verticals (Creator Hub, Ops Intelligence, and future modules).

## Capabilities (Phase 3)

- **Cross-domain retrieval** — query across creator and ops knowledge
- **Contextual memory** — persistent context that spans sessions and verticals
- **Retention policies** — configurable data lifecycle management

## Architecture

```
knowledge-fabric/
  src/
    cross-domain/     # Cross-vertical retrieval and linking
    retention/        # Data lifecycle and retention policies
    index.ts          # Module exports
```
